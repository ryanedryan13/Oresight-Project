# ORESIGHT

ORESIGHT (Optimal Replenishment Engine & Inventory Operations Nexus) is a single-file HTML calculator for inventory planning across multiple warehouses.

## What it does
- Calculates optimal reorder points
- Calculates reorder quantities
- Calculates safety stock levels
- Supports multi-warehouse inventory analysis

## How to use
1. Download or clone the repository
2. Open the HTML file in any modern browser
3. Enter your inventory inputs
4. Review the calculated replenishment outputs

## Requirements
- No installation
- No build tools
- No server
- Just a modern web browser
